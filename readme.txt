こちらは2018年の京都大学11月祭にて頒布いたしました
「敵を避ける遊戯」の説明書です。

#概要
このゲームは、
何者かによって体をSF的に武装された鳥型宇宙人が自身の体を狙う何者かの襲撃をよけ続けるゲーム
です。
フィールド上で主人公を動かし何者かの手から逃げ回りましょう！

#起動方法
(こちらのゲームはWindowsにしか対応しておりません。申し訳ございません。
(Pythonの環境をお持ちでしたらmain.pyを実行することで遊ぶことが可能かと思いますが、
動作確認等が出来ておりません))
"teki_wo_yokeru_game"を実行してください。

追記：Visual C++ 2015再頒布可能パッケージがインストールされていないと動作しません。
非常にお手数ですが、下記のリンクからインストールをお願いします。
https://www.microsoft.com/ja-jp/download/details.aspx?id=48145

#基本操作
方向キー上：選択肢移動、ジャンプ
　　　　下：選択肢移動、穴を掘る
　　　　左・右：移動
S：決定、穴を掘る　など
W：ジャンプ
A,D：ダッシュ(ワープ)
Enter：決定(タイトル画面)、ポーズ(ゲーム画面)

ゲーム中の操作はポーズ画面(Enterを押す)でも確認いただけます。
